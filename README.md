# IPERC para Pymes – Proyecto Final Cloud Computing & DevOps

Sistema web para el registro de evaluaciones de riesgos laborales (IPERC) en pymes peruanas.

## Stack
- Node.js 18 + Express
- Docker
- AWS: ECS Fargate, DynamoDB, ECR, ALB, VPC
- Terraform (IaC)
- GitHub Actions (CI/CD)
